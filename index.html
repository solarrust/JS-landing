Зачем изучать JavaScript?

JavaScript — единственный язык программирования, который работает в браузерах на компьютерах, телефонах, планшетах и смарт-тв. Можно смело назвать JavaScript самым практичным и необходимым в арсенале веб-программиста вне зависимости от того, на какой платформе реализован сайт (PHP, Ruby, Java). Мы в «Нетологии» спроектировали этот курс так, чтобы каждый выпускник получил крепкую базу для использования языка во всех возможных сферах и смог реализовать на JavaScript любой алгоритм. Останется лишь найти интересную задачу.
